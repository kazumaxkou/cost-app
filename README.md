# 製造コスト見積もりツール

## 🔗 アプリURL

https://cost-app-dq28sj96vjjcqfaxanesib.streamlit.app/

---

## 📌 概要

製品タイプと加工時間を入力することで、製造コストを自動で算出するWebアプリです。
前職での見積業務の経験をもとに、作業効率化を目的として開発しました。

---

## ⚙️ 主な機能

* 製品タイプ選択（A / B）
* 加工時間入力
* 自動コスト計算
* 10時間以上で割引適用
* 計算結果のCSV保存
* 売上データのグラフ表示

---

## 🛠 使用技術

* Python
* Streamlit
* pandas

---

## 💡 工夫した点

* 入力操作をシンプルにし、誰でも使いやすいUIにした
* 計算結果をCSVに保存し、データ蓄積を可能にした
* グラフ表示により売上の可視化を実現
* 前職の業務フローを意識したロジック設計

---

## 🚀 今後の改善案

---

## 👤 制作背景

前職の製造業において、見積作業に時間がかかる課題を感じていました。
その経験から、簡単にコスト算出ができるツールを作成し、業務効率化を目指しました。

---

## 📷 使い方

1. 製品タイプを選択
2. 加工時間を入力
3. 「計算」ボタンをクリック
4. コストが表示され、データが保存されます
